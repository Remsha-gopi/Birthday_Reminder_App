# Birthday Reminder App

A desktop application built with **Python** and **Tkinter** to manage and track birthdays. The app allows users to add birthdays, view them in a structured table, highlight today’s birthdays, and receive notifications for upcoming birthdays. Data is stored in a **JSON file** for persistence.

---

## Features

- Add and store birthdays with **name** and **date**.
- View all birthdays in a **table format**.
- Highlight **today’s birthdays**.
- Popup notifications for **upcoming birthdays**.
- Persistent storage using a **JSON file**.
- Simple and intuitive **GUI**.

---

## Technologies Used

- Python 3  
- Tkinter (GUI)  
- JSON (data storage)  
- `datetime` module (date handling)
